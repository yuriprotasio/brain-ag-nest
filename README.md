## Descrição

Brain AG, sistema crud de agricultura desenvolvido em nestjs e typeorm.

## Instalação

```bash
yarn install
```

## Iniciando a aplicação

```bash
yarn typeorm migration:run

yarn start
```

## Testes Unitários

```bash
Crie um banco de dados no postgres com o nome brain_ag_test

yarn test
```
